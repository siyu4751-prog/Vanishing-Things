# Vanishing-Things
# Fading Everyday｜消逝日常档案

> 记录那些尚未完全消失，却已经很少被我们注意到的物品、场景与生活习惯。

# 项目介绍

科技和生活方式不断变化，一些曾经随处可见的东西正在逐渐退出日常生活。它们可能是一张纸质车票、一部公共电话、一封手写信，也可能是校园里的旧公告栏、街边的修鞋摊，或者一句只有老一辈还在使用的方言。
Fading Everyday 希望通过照片、文字和个人记忆，为这些正在消失的日常事物建立一份公开档案。

# 收录标准

本项目主要收录：
 曾经被广泛使用，现在已经逐渐少见的物品。正在被新技术或新生活方式取代的场景。 具有地域、校园或家庭特色的生活记忆。能够通过照片、资料或个人经历进行说明的内容

# 如何参与

如果你也发现了正在消失的日常事物，可以通过 Issue 提供：

1. 物品或场景名称
2. 拍摄地点和时间
3. 原创照片
4. 与它有关的故事
5. 你认为它正在消失的原因
